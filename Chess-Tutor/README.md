# Chess Tutor

This is a Python-based chess application with a graphical interface.

## How it works
- The program displays a chess board using images.
- Each chess piece is represented with image files.
- The user can interact with the board and play moves.
- The game logic is handled across multiple Python files.

## How to run
python main.py

## Requirements
- Python 3
- tkinter

## Files
- main.py → runs the program
- ai.py → handles basic game logic
- board_ui.py → manages the board display
- images → contains chess piece images
