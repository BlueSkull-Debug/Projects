# Language Bias Analyzer

This is a web app that analyzes text for bias and emotional language.

## How it works
- The user enters article text or a URL.
- The program checks for biased or emotional wording.
- It shows a score and highlights the text.

## How to run
streamlit run main.py

## Requirements
- Python 3
- streamlit
- requests
- pandas
- beautifulsoup4

## File
- main.py
