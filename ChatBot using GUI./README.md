# Chatbot (GUI)

This is a simple chatbot built with Python and a GUI.

## How it works
- The user types a question into the app.
- The chatbot searches for the closest matching question.
- It returns the best answer from a dataset.

## How to run
python main.py

## Requirements
- Python 3
- tkinter
- pandas
- gspread
- fuzzywuzzy

## File
- main.py
