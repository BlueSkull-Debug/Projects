# Rock Paper Scissors (GUI)

This project is a simple Rock Paper Scissors game built using Python and Tkinter for the graphical user interface.

## 🧠 How it works
- The user selects Rock, Paper, or Scissors by clicking a button.
- The computer randomly selects its choice.
- The program compares both choices and determines the winner.
- Scores for both the user and the computer are updated and displayed.

## 🖥️ Features
- Interactive GUI using Tkinter
- Random computer choice
- Score tracking system
- Displays results after each round
